### 矩阵和向量的乘法与吧矩阵看做向量的函数

```

    矩阵和向量相乘
        matirx(常量系数) * 列向量 (参数) = 方程组常量

        （1， -0.2， 0.1， 0.5） * （xit, xe, xm, xh) = (100, 50, 60 ,666)

        1. 矩阵的列数必须和向量 u 的元素个数一致 ！
        2. 矩阵 A 的行数没有限制
            行数为 1 则变为 向量点乘向量

    T * vec = vec2

    矩阵 T 实际上将向量 vec 转换成了向量 vec2
    可以把矩阵理解为 向量的函数 ！
```

### 矩阵与矩阵的乘法

```
    矩阵与向量相乘 -> 矩阵函数对一个向量做函数运算 -> 产生一个向量

    矩阵与矩阵相乘 -> 矩阵函数对于一组向量做函数运算 -> 产生一组向量 -> 产生另外矩阵

    A * B = T
    
    矩阵 A 的列数必须和矩阵 B 的行数一致 ！
    A 是 m*k 的矩阵； B 是 k * n 的矩阵，则结果矩阵为 m * n 的矩阵

        矩阵乘法不遵守交换律 A * B != B * A （可能不合法） 即使可以相乘， 结果也不一样
```

### 矩阵乘法的性质和矩阵的幂

```
    1. 矩阵乘法不遵守交换律！ A * B != B * A

    2. 矩阵乘法遵守:

    
    矩阵的幂：
```

### 矩阵的转置

> 矩阵的转置：行变成列；列变成行 A = aij -> AT = aji

```
    矩阵转置的性质
        （AT)T = A
        (A+B)T = AT + BT
        (k * A)T = k * AT
        (A * B)T = BT * AT
        
```
