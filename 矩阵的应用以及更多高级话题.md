## 矩阵的应用及其高级话题

### 更多变换矩阵
1. 矩阵表示变换 (向量的函数)

```
    (二维空间)
    让每个点的横坐标扩大 a 倍， 纵坐标扩大 b 倍

    T = a 0     -> T * vec(x, y) = vec(ax, by)
        0 b     

    让每个点关于 X 轴翻转

        T * vec(x, y) - > vec(x, -y)  -> 设 T = a b
                                                c d
            代入方程式得：
                T = 1 0
                    0 -1

    让每个点关于 Y 轴翻转
        同理得：
                T = -1 0
                     0 1
    
    让每个点关于原点翻转 （X 轴， Y 轴均翻转）
        同理得：
                T = -1 0    ==  Tx(Ty * T) (现在 y 轴上翻转，再到 y 轴上翻转)
                     0 -1

    错切
        沿 X 方向错切

            T * vec(x, y) = vec(x + ay, y)
        
        沿 Y 方向错切

            T * vec(x, y)  = vec(x, y + ax)
```
